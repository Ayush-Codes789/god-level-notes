# god-level-notes
1. **Clone the Repository**
   ```bash
   git clone https://github.com/Ayush-Codes789/god-level-notes.git
   cd https://github.com/Ayush-Codes789/god-level-notes/blob/main/app.py
   ```

2. **Install the dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the Streamlit app**
    ```bash
    streamlit run app.py
    ```

4. **Configure API Keys**
   - Enter your API keys in the sidebar
   - All three keys ( Gemini) are required for full functionality
